県道の文字の書き換えについて
	県道の看板には3つの文字の層があります。
	上から順に説明していきます
		一つ目が"県道"と書いてあるところ
		二つ目が県道何号かを表すところ
		三つ目がどこの県道かを表すところ
	以上の層があります。
	一つ一つ書き換えについて解説していくと、
		一つ目は書き換えなくていいです。(二つ目が文字の場合は書かない場合もあるらしい?(後述))
		二つ目は県道何号かを表すので、自由に数字を変えて遊びましょう。
		三つ目はどこの県道かを表すので、看板がある場所の県名を書いておきましょう。(二つ目が文字の場合は書かない場合もある)
	以上の基本以外にも、特殊な表示の仕方があります。
		二つ目を路線番号ではなく、路線名称で書くの看板もあります。(千葉に多いらしい?)(rrs_hojotakizawaが一応作ってみたものです)
		看板ではなく、札?のようなものもあります。(山梨県道810号雨畑大島線の見神の滝にある駐車場付近にあります(見つけるのは結構難しいです。)
		路線番号に加えて、号をつけるものもあるそうです。(https://commons.wikimedia.org/wiki/File:Hexa22.JPG#/media/ファイル:Hexa22.JPG にあります)

国道の文字の書き換えについて
	国道の看板には特殊なものがほぼありません。
	書き換える場所は路線番号の部分だけです。

林道の文字書き換えについて
	林道のもので書き換えるのは路線名と管理している県または市区町村の表示,起終点どちらかの表示だけです
	路線名は"線"を含めて3文字がおすすめです。
	管理県市区町村は上の路線名よりも少し小さめにします。
	起終点のものはすでにある(\temp\rindoyamanakaendまたは\temp\rindoyamanakastart.png)ものから抜き取ってやるといいとおもいます。
	(民間保有の場合は、管理市区町村の表示がありません)

幅員の市区町村書き換えについて
	fukuin_kakikae.pngに書いてあるように12pxで文字を打ち込むだけです。
	水色のところを透過にして保存し、rrs_[ほかの標識と被らない文字列]にすることで使えます。
	駅名標バージョンは書き換えたのち、jsonのファイル名を変えるだけです。

キロポストのキロ程書き換えについて
	普通に文字を変えるだけ。基本的には0.5km単位でやる。

駅名標の書き換えについて
	\assets\minecraft\textures\signboardのakikawa_kihon.pngを基本にしてください。
	書き換えるものとしてはakikawa_muziL.pngとかがあるのでそれを基本のやつに従って書くだけです。jsonは内容を以下に貼っておくので書き換えが必要なところだけ書き換えてぶち込んでください。
-------------------------------------------------
{
    "texture": "書き換えたもののファイル名(.png拡張子なしのファイル名の形式で設定する。ディレクトリのパスは不要。)(恐らく英数字のみ対応)",
      "backTexture": 1,
    "height": 0.75,
    "width": 2.25,
    "depth": 0.375,
    "frame": 1,
    "animationCycle": 1,
    "color" : 0,
      "lightValue": 4
  }
-------------------------------------------------
一つ一つ説明をすると、

{ <-jsonを開始するよっていうやつ。 htmlで言う<html>のこと
    "texture": "akikawa_kihon", <-何というファイル名の駅名標を指定するか
      "backTexture": 1, <-0か1で指定。1だとpng画像の横幅の半分が後ろの駅名標の表示になる。
    "height": 0.75, <-駅名標の高さ。1にすると1mきっかりの高さの駅名標になる。
    "width": 2.25, <-駅名標の幅。 1にすると1mきっかりの横幅の駅名標になる。
    "depth": 0.375, <-駅名標の奥行。 1にすると1mきっかりの奥行きの駅名標になる。 height,width,depthを全部1にすると1mの普通のブロックが出来上がる。
    "frame": 1, <-コマ数/フレーム数の指定 ここを縦のピクセル数に指定すると駅名標がゆっくり回転する。 がちのアニメーションはめんどくさいのであまりおぬぬめしない
    "animationCycle": 1, <-よくわかってない。 でもアニメーションしないときは1でさせたいときは5にしてる。 たぶん20tickに何回コマを回すかの設定
    "color" : 0, <-駅名標の横のカラーを指定する。 色の計算方法がわかってない。
      "lightValue": 4 <- -16~15の間で指定する。 -16だとちかちかする。 -のものはRSを入れてる時だけ明るくなる。 通常の1~15は常に輝く。 数字は明るさレベルの指定。
  } <-jsonを終了するよっていうやつ。htmlで言う</html>のこと
-------------------------------------------------
	ちなみに、青い奴の方も\assets\minecraft\textures\signboard\blue_sign_noneを書き換えて、\assets\minecraft\textures\signboard\SignBoard_blue_sign_noneの内容をコピーしてファイル名変えるだけでいけます。-

様々なものの文字テンプレート/フォントについて

mojisankou.pngが県道標識の文字の参考です。(出典:https://web.archive.org/web/20070228084944im_/http://www.mlit.go.jp/road/sign/kijyun/kukaku/bpkukaku02/ssd005.gif)
国道標識は自作または色々なフォントの改造,作者が撮影した画像からの抽出です。
LED発車案内表示板は西武線の物を使用しています(ドット数がほぼ同じ！ すごい！)(どうこんしてない)
駅名標,隧道の銘板,幅員狭しは"UDEV Gothic 35LG"というフォントを使用しています。
制限標識はRTM本体の制限標識のフォントを使用しています。(一部はこれを改良した物を使用しています。)
停車位置目標1~10はRTM本体の停車位置目標のフォントを使用しています
停車位置目標11~14はRTM本体の停車位置目標のフォントを改良した自作フォントを使用しています。
村道標識はすべて自作フォントを使用しています。
林道標識は メイリオ を使用しています。
キロポストのキロ程部分は"UDEV Gothic NFLG",'[ｵﾆｷﾞﾘ]より'と'km'の部分は自作フォントを使用しています。
アドレナリンは自作フォントを使用しています

書いてないもののフォントは忘れました。土下座
